2. package.json (root) mein ek hi dev script rakho
Tumhari monorepo mein pnpm workspaces hain. Har app (API, admin, vendor, rider) ke paas ek dev script honi chahiye. Root script yeh rakho:

json
"scripts": {
  "dev": "pnpm --parallel -r run dev",
  "start": "pnpm --filter @workspace/api-server start"
}
pnpm --parallel -r run dev ka matlab: saare workspaces ki dev script ek saath parallel mein run hogi, aur jab tak koi process alive hai, Replit ka container bhi chalta rahega.

Agar tumhare packages ke naam @workspace/admin nahi hain to unhe apne workspace name ke hisaab se adjust kar lena.

3. .npmrc file root folder mein banayein
Yeh pnpm ko har platform par sahi tarah dependencies install karne mein help karega.

ini
shamefully-hoist=true
strict-peer-dependencies=false
auto-install-peers=false
child-concurrency=5
network-timeout=100000
4. Backend (api-server) mein CORS dynamic karo
server.js / app.js mein jaakar hardcoded origin list hatao aur yeh code lagao. Ab koi bhi localhost, Replit, ya Codespaces ka dev URL automatic allow hoga.

javascript
const cors = require('cors');

app.use(cors({
  origin: function (origin, callback) {
    if (
      !origin ||
      origin.startsWith('http://localhost') ||
      origin.startsWith('http://127.0.0.1') ||
      origin.includes('.replit.dev') ||
      origin.includes('.replit.app') ||
      origin.includes('.github.dev')
    ) {
      callback(null, true);
    } else {
      callback(new Error('CORS not allowed'));
    }
  },
  credentials: true
}));
